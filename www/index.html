Synopsis

Figure 1
^That's the SUN!!!

The Heliostat is a Solar Imaging platform that was designed and implemented by UNCo Physics students in the mid-90s.

This project is actively seeking interested students who want to gain experience in mechanical and electrical design, software development, project management, and documentation. Contact Daniel Packard or Maurice Woods to find out more.

To make the project more accessible to new users, all existing hardware design, software, and documentation (including the source for this web page) have been compiled into a github repository. Visit the page on github (Heliostat on Github) to find out more.

You can clone your own copy of the to start using/contributing to the project right away! For example, on Linux you would issue the following command from the command line:

$ git clone http://github.com/pack3754/unco-heliostat.git

Details

To achieve the project goals, students and advisors will conduct the following tasks:

  • review the current state of hardware and software
  • examine possible modifications to the hardware platform to address issues such as weather proofing, communication interface (ethernet/wi-fi), and power systems
  • implement any fixes or modifications to the hardware platform
  • design a client-server software interface to control the motors from any remote computer
  • and implement various GUI front-end utilities to make controlling the Heliostat accessible for users who are unfamiliar with the project

Many of these tasks can be conducted independently, with a final integration phase taking place towards the end of the project. The integration phase will provide students with a good opportunity to review and edit the documentation that has been developed by their peers throughout the course of the project.
